# IntelliGrade

**IntelliGrade** is an AI-Assisted Examination Script Evaluation System developed as a Software Engineering practicum project at IUBAT.

## Objective

To assist instructors in evaluating descriptive answer scripts using AI while maintaining full instructor control over the final grading process.
