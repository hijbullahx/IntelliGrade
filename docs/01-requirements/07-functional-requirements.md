FR-01

The system shall authenticate users.

FR-02

The system shall allow teachers to create examinations.

FR-03

The system shall allow uploading answer scripts.

FR-04

The system shall allow defining question-wise rubrics.  

FR-05

The system's AI Evaluation Engine shall generate rubric-based scoring suggestions.

FR-06

The system shall construct structured prompts from grading rubrics and answer text.

FR-07

The system shall support multiple LLM providers through a provider layer.

FR-08

The system shall normalize AI-generated scores into a consistent format.

FR-09

The system shall calculate a confidence score for each AI-generated evaluation.

FR-10

The system shall prevent the automatic publishing of grades without explicit teacher approval.
